Training Modules

HTML

HTML stands for Hyper Text Markup Language. It is the most widely used language on Web to develop web pages. This module teaches you everything about HTML and it is easy to learn.

CSS

CSS is a language that describes the style such as fonts, colors and spacing to an HTML document. This module will teach you CSS from basic to an advanced level.

JavaScript/ES6

JavaScript is the programming language of HTML and the Web. It is complimentary to and integrated with Java. This module will make you learn JavaScript from basic to an advanced level.

MongoDB

MongoDB is a general purpose, document-based, distributed database built for modern application developers and for the cloud era. No database is more productive to use.

Node.js

Node.js is an open source server environment that uses JavaScript on the server, asynchronously and runs on various platforms.

React

A JavaScript library for building user interfaces.React makes it painless to create interactive UIs.

Django

Django is a Python Web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of Web development.

Redux

Redux is an open-source JavaScript library for managing application state. It is most commonly used with libraries such as React or Angular for building user interfaces.
